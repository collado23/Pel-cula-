# StreamVault Pro

Aplicación web para visualizar y gestionar listas de canales IPTV (M3U).

## Características:
- Carga automática de listas M3U.
- Organización automática por categorías.
- Sistema de favoritos.
- Limpiador de enlaces caídos.

## Cómo usar:
Haz clic en "Cargar Lista" para procesar el archivo `canales.m3u` configurado en el sistema.
